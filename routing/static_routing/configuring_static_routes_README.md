# Static Routing Lab 

This Packet Tracer lab demonstrates basic static route configuration between three routers. 
Includes:
- IP addressing
- Setting default gateways
- Configuring and enabling router interfaces 
- Configuring static routes
- 
- Route verification with show ip route
- Interface verification with show ip int brief
- 
- sucessful pings between 2 pcs on 2 different networks via configured static routes
